## Deskripsi
Repository ini berisi eksperimen dan preprocessing untuk Proyek Akhir kelas Membangun Sistem Machine Learning (MSML) Dicoding.

## Dataset
- **Nama**: Iris Dataset
- **Sumber**: Scikit-learn
- **Ukuran**: 150 baris × 5 kolom
- **Fitur**: sepal length, sepal width, petal length, petal width
- **Target**: spesies bunga (Setosa, Versicolor, Virginica)
- **Tipe Problem**: Klasifikasi multiclass
